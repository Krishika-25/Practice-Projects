# Smart To-Do App

A simple web app to keep track of your tasks.  

---

## Features

- Add tasks  
- Edit tasks  
- Delete tasks  
- Mark tasks as completed  
- Clear all completed tasks  
- Shows total, pending, and completed tasks  

---

## How to Use

1. Type your task in the input box.  
2. Press **Add** or **Enter**.  
3. Click a task to mark it completed.  
4. Click ✏️ to edit or ✖ to delete a task.  
5. Click **Clear Completed** to remove finished tasks.  

---

## Tech Used

- HTML  
- CSS  
- JavaScript  
- `localStorage` for saving tasks  

---

## Notes

- Tasks are saved in the browser so they stay even after refreshing.  
- Works on any modern browser.
